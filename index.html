Spring Sandals

A lightweight approach to building applications with the Spring Framework

Overview

Spring Sandals is a developer's guide for building lightweight applications using Spring Framework.

We’ve seen many teams move away from Java and Spring in search of lighter-weight languages and frameworks, only to realize there’s no silver bullet. In response, we maintain a collection of examples of how Spring developers can achieve the advantages of a light-weight framework while taking advantage Spring’s strengths.

The move often feels AI-driven, led by Python and libraries like Flask, BeautifulSoup, and LandChain. Developers then attempt to replicate the aspects of classic Java stacks in their new stack, only to find that building component-based architectures can be a real challenge in some languages and frameworks. To support this transition to AI, we’ve created the AI Discovery Agent project—built on top of the embedded-server template—to demonstrate how Spring continues to shine.
