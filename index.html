Spring Sandals

A lightweight approach to building applications with the Spring Framework

Overview

Spring Sandals is a developer's guide for building lightweight applications using the Spring Framework.

We’ve seen many teams move away from Java and Spring in search of lighter-weight languages and frameworks - only to realize there’s no silver bullet. In response, we’re sharing a collection of examples of how Spring developers are adapting to this shift while continuing to take advantage of the framework’s strengths.

The move often feels AI-driven, led by Python and libraries like Flask and BeautifulSoup. Developers then make attempts to revive the “good parts” of classic Java stacks in their new stack - only to find that building component-based architectures can be a real challenge in other languages and frameworks. To support this transition to AI, we’ve created an AI Discovery Agent project - built on top of the embedded-server template to demonstrate how Spring can still shine in this evolving landscape.
